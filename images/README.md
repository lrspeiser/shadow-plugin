# Extension Icon

Place a `icon.png` file here (128x128 pixels).

The icon should represent "watching" or "monitoring" - suggestions:
- An eye symbol
- A radar/scanning symbol
- A shield with eye
- Combination of code brackets + eye

## Creating the Icon

1. Use any graphics tool (Figma, Sketch, Photoshop, etc.)
2. Design at 128x128px
3. Use Shadow Watch brand colors if available
4. Export as PNG
5. Name it `icon.png`
